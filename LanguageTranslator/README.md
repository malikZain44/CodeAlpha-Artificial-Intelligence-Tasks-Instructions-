# 🌍 AI Language Translator Pro 

### *Neural Machine Translation with Premium Glassmorphism UI*

AI Language Translator ek modern aur intuitive web application hai jo text ko instantly mukhtalif languages mein translate karta hai. Is project ko **Python** aur **Streamlit** ka istemal karte hue design kiya gaya hai, jisme ek professional **Glassmorphism UI** ka touch diya gaya hai taake user experience behtareen ho.

---

## ✨ Key Features

* **🚀 Instant AI Translation:** Neural network-based translation jo accurate aur fast results deti hai.
* **💎 Glassmorphism UI:** Frosted glass effect, blur backgrounds, aur sleek animations ke saath ek modern dark-theme interface.
* **🔊 Text-to-Speech (TTS):** Translated text ko sunne ke liye built-in voice support, jo pronunciation mein madad karta hai.
* **📱 Responsive Design:** Har tarah ki screen (Desktop, Tablet, Mobile) par perfectly adjust hone wala layout.
* **⚡ Session Persistence:** `st.session_state` ka istemal kiya gaya hai taake audio play karte waqt aapka translated data reset na ho.
* **🌐 Multi-Language Support:** English, Urdu, Hindi, French, German, aur Spanish jaisi badi languages ka support.
