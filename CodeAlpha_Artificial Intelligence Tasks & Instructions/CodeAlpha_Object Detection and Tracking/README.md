# 👁️ AI Object Detection & Tracking - Task 4

## Project Overview
[cite_start]Developed a real-time object detection and tracking system using **YOLOv8** as part of my **CodeAlpha** internship[cite: 13, 19]. This project goes beyond simple detection by implementing unique tracking IDs and a counting logic for objects crossing a defined boundary.

## Key Features
- **Model:** YOLOv8 (Medium version) for a balance between speed and accuracy.
- **Tracking:** Implemented BoT-SORT / ByteTrack for persistent object IDs.
- **Analytics:** Real-time counting of objects (In/Out) with a graphical dashboard overlay.
- **Dynamic Region:** Custom counting line established to monitor traffic flow.

## Tech Stack
- Python, OpenCV, Ultralytics YOLOv8, Matplotlib.
